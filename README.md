# KoGaMa ModelAPI

ModelAPI Core for KoGaMa. This library provides the core data structures and utilities for handling KoGaMa models, specifically focusing on cube-based representations.

## Features

- **Cube-based Model Representation**: Defines `Cube` and `ModelInfo` classes to represent 3D models.
- **Coordinate Handling**: Includes `IntVector` and `PositionConverter` for managing positions and vector conversions.
- **Serialization Interfaces**: Provides `ISerializer` for implementing custom model serialization.

## Project Structure

- `MauryDev.KoGaMa.ModelAPI`: The core library targeting .NET Standard 2.0.
- `TestProject`: Unit tests for the core library.

## Getting Started

### Prerequisites

- .NET SDK (compatible with .NET Standard 2.0)

### Installation

You can include the project in your solution or package it as a NuGet library.
